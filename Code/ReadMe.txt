
The structure of the code is as follows:

Folders:
CSS folder-> Contains all the css files for the system
IMG folder -> Contains all the logos and images used througout the system.
JS folder -> contains APP.JS,which defines all the modules; and the following subfolders: Controllers, Services, Vendor
	Controllers subfolder: Contains the controllers used in the system
	Services subfolder: Contains DataRequest, which handles all the request to the database.
	Vendor subfolder : Contains all the plug-ins and the external libraries saved locally in the system
Scripts folder: Contains all the scripts for testing.
Uploads folder: Contains a list of subfolders representing the categories,each subfolder contains the documents uploaded to the system.
Views folder: Contains the html pages for each implementation.

Index.HTML : Main page
